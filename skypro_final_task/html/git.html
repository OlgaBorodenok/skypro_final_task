Git

  • git init — инициализирует репозиторий Git в выбранной папке.
  • git clone — скачивает удаленный репозиторий и создает его локальную копию.
  • git status — выводит текущее состояние рабочей директории и индекса: какие файлы изменены, добавлены, неотслеживаемые и т.д.
  • git add — выбирает конкретный файл для включения в будущий коммит, добавляя его в индекс.
  • git commit — фиксирует изменения, добавленные в индекс, в репозитории.
  • git branch — создает новую ветку, оставаясь при этом в текущей.
  • git checkout — осуществляет переход на указанную ветку.
  • git push — отправляет локальные изменения в удаленный репозиторий.
  • git pull — получает изменения из удаленного репозитория и автоматически сливает с локальной веткой.
